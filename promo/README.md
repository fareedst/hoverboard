# Promotional assets

## Chrome Web Store images

Assets for the Hoverboard extension listing.

| File | Size | Use |
|------|------|-----|
| `hoverboard-promo-small-440x280.png` | 440×280 px | **Required** – homepage, category, search results |
| `hoverboard-promo-marquee-1400x560.png` | 1400×560 px | Optional – marquee carousel when featured |

- Full bleed, no borders or padding.
- Layout: left = brand block (icon + “Hoverboard” + “Bookmarks, organized.”); right = simple “after” metaphor (tidy bookmarks, checkmarks).
- Colors: cyan/teal accent (#00BCD4). Background contrasts with the store’s light gray (no white or light gray) so the tile stands out. See [Chrome Web Store image guidelines](https://developer.chrome.com/docs/webstore/images/).
